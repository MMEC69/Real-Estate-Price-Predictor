# Real Estate Price Prediction Web App

This project is a web application that predicts real estate prices using a linear regression model built with Python's scikit-learn library. The application provides a user-friendly interface to input property features and receive an estimated price prediction.

## Features

* **Price Prediction:** Predicts real estate prices based on user-provided features.
* **Linear Regression Model:** Utilizes a scikit-learn linear regression model for predictions.
* **User-Friendly Interface:** Built with HTML, CSS, and JavaScript for a seamless user experience.
* **Python Flask Server:** Uses a Flask server to handle backend logic and requests.
* **Interactive Input:** Allows users to input relevant property details through a web form.
* **Clear Output:** Displays the predicted price to the user.

## Technologies Used

* **Frontend:**
    * HTML
    * CSS
    * JavaScript
* **Backend:**
    * Python
    * Flask (web server)
    * scikit-learn (sklearn)
* **Model:**
    * Linear Regression
